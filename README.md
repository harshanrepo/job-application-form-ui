📄 Job Application Form UI

A modern job application form built using HTML and CSS. This project focuses on clean layout, form styling, and visual feedback using validation states.

---

🚀 Live Demo

👉 https://harshanrepo.github.io/job-application-form-ui/

---

📌 Features

* Clean and structured form layout
* Glassmorphism (blur background effect)
* Custom styled radio buttons
* Hover effects on button

---

🛠️ Built With

* HTML5
* CSS3 (Flexbox, backdrop-filter, pseudo-classes)

---

## 📚 What I Learned

* Styling forms using CSS
* Using pseudo-classes like :focus, :valid, :invalid
* Creating glassmorphism UI effects
* Improving user experience with visual feedback

---

🔧 How to Run Locally

1. Clone the repository
2. Open `index.html` in your browser
